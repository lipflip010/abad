# abad
Simple python webserver

### Deployment
    sudo aptitude install python3-venv
    sudo aptitude install python3
    git clone git@github.com:lipflip010/abad.git
    cd abad
    python3 -m venv ./env
    cd env/bin
    source activate
    cd ../../../abad
    pip install -r requirements.txt
